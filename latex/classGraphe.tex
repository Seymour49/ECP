\subsection{Detailed Description}
Cette classe représente un graphe dont le fichier source (format D\-I\-M\-A\-C\-S) est passé en paramètre au constructeur. Une matrice de booléens taille N$\ast$\-N (où N est le nombre de sommets est créée pour représenter ce graphe. Ainsi M\mbox{[}u\mbox{]}\mbox{[}v\mbox{]} = 1 ssi (u,v) € E (ensemble d'arcs) 0 sinon.
